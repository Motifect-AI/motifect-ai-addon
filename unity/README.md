# Motifect Motion — Unity

Unity 2021.3+ UPM package.

## Install

**Window → Package Manager → + → Add package from git URL:**

```
https://github.com/Motifect-AI/motifect-ai-addon.git?path=/unity/MotifectMotion
```

Or copy [`MotifectMotion`](MotifectMotion) to `YourProject/Packages/MotifectMotion/`.

## Use

**Window → Motifect → Motion Generator** → API key + prompt → **Generate Motion**.

Imports FBX to `Assets/Motifect/Generated/`.

## Retargeting

Humanoid import / Animation Rigging, or [motifect.io retarget](https://motifect.io).
