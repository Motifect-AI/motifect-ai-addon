# Motifect Motion — Unity

## Overview

**Free package.** Generates human motion from English text prompts using the [Motifect API](https://motifect.io) and imports FBX files into your project.

## Requirements

- Unity **2022.3 LTS** or newer
- Internet connection
- [motifect.io](https://motifect.io) account
- API key (`mk_live_...`) from **Developer → API Keys**
- **Paid credits** on your account (each generation consumes credits)

## Install

**Package Manager → + → Add package from git URL:**

```
https://github.com/Motifect-AI/motifect-ai-addon.git?path=/unity/MotifectMotion
```

Or copy this folder to `YourProject/Packages/com.motifect.motion/`.

## Setup

1. Open **Window → Motifect → Motion Generator**
2. Enter your API key
3. Enter an English prompt (10–180 characters)
4. Choose duration and model
5. Click **Generate Motion**

Imported assets appear in `Assets/Motifect/Generated/`.

## AI disclosure

Motion is generated by the Motifect cloud service (AI-based text-to-motion). This package is a client only; generation runs on motifect.io servers.

## API key storage

Your API key is stored in Unity **EditorPrefs** on your machine only. It is not included in builds.
