# Motifect Motion — Unreal Engine

## Overview

**Free plugin.** Generates human motion from English text via the [Motifect API](https://motifect.io) and imports FBX into your project.

## Requirements

- Unreal Engine **5.x**
- **Python Editor Script Plugin** (enabled automatically by this plugin)
- Internet connection
- [motifect.io](https://motifect.io) account + API key + **paid credits**

## Install

Copy the `MotifectMotion` folder to `YourProject/Plugins/MotifectMotion/` and restart the editor.

## Setup

1. **Tools → Motifect → Open Config Folder**
2. Edit `Saved/Motifect/config.json` — set `api_key` to your `mk_live_...` key
3. **Tools → Motifect → Generate Motion**

Or set environment variable `MOTIFECT_API_KEY`.

## Config fields

See `Config/motifect_settings.example.json`:

| Field | Description |
|-------|-------------|
| `api_key` | Motifect API key |
| `prompt` | English motion description |
| `duration_seconds` | 2–10 |
| `model_key` | `motifect-v3-fast`, `motifect-v3`, or `kimodo-human` |
| `import_destination` | Content path, e.g. `/Game/Motifect/Animations` |

## AI disclosure

Motion is generated by the Motifect cloud service (AI text-to-motion). This plugin is a client only.
