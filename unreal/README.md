# Motifect Motion — Unreal Engine

Unreal Engine 5.x plugin (Python Editor Script Plugin required).

## Install

Copy [`MotifectMotion`](MotifectMotion) into `YourProject/Plugins/MotifectMotion/`, enable **Python Editor Script Plugin**, restart.

## Use

1. **Tools → Motifect → Open Config Folder** — set `api_key` in `Saved/Motifect/config.json`.
2. **Tools → Motifect → Generate Motion**

See `MotifectMotion/Config/motifect_settings.example.json` for all settings.

## Retargeting

Retarget in Unreal or via [motifect.io retarget](https://motifect.io).
