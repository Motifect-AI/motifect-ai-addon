# Motifect Motion — Blender

Generate human motion from text in Blender 3.6+ (tested on 4.4).

## Install

1. Download **[motifect_ai_motion.zip](../../releases/latest)** from the repo Releases page.
2. **Edit → Preferences → Add-ons → Install…** → select the zip.
3. Enable **Motifect Motion**, paste API key (`mk_live_...`).
4. 3D Viewport **N** → **Motifect** → **Generate Motion**.

## Reinstall

Quit Blender and delete `%APPDATA%\Blender Foundation\Blender\<version>\scripts\addons\motifect_ai_motion`, then install again.
