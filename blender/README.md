# Motifect Motion — Blender

Generate human motion from text in Blender 4.2+ (extension manifest) or 3.6+ (legacy zip).

**Free add-on** — requires motifect.io account, API key, and paid credits per generation.

## Install

1. Download **[motifect_ai_motion.zip](../../releases/latest)** from Releases.
2. **Edit → Preferences → Add-ons → Install…** → select zip.
3. Enable **Motifect Motion**, paste API key (`mk_live_...`).
4. Enable **System → Allow Online Access** (required for API).
5. 3D Viewport **N** → **Motifect** → **Generate Motion**.
