# Marketplace submission guide

Motifect Motion add-ons are **free**. Revenue is from **motifect.io credits** only.

Target platforms (not Blender official Extensions):

| Platform | Category | Status after code prep |
|----------|----------|------------------------|
| **Superhive** (Blender) | Add-on zip | Ready for Creator apply + listing |
| **Unity Asset Store** | Tools / UPM | Ready for Publisher + listing |
| **Epic Fab** (Unreal) | Tools & Plugins | Ready for listing + Fab zip |

## Listing copy (paste at top of every product page)

```
FREE add-on / plugin — no purchase price.

Requires:
• motifect.io account (free signup)
• API key from Developer → API Keys
• Paid credits per generation (see motifect.io pricing)
• Internet connection

Motion is generated by the Motifect cloud API (AI text-to-motion).
This package is an editor client only; generation runs on motifect.io servers.

Terms: https://motifect.io/terms
Docs: https://motifect.io/en/docs
Support: https://github.com/Motifect-AI/motifect-ai-addon/issues
```

### Unity — AI description field (required)

```
Uses the Motifect cloud API for AI-based text-to-human-motion generation.
This package is a Unity Editor client; no AI models are bundled locally.
Generated motion is downloaded as FBX and imported into the project.
```

## Superhive (Blender)

**Deliverable:** `blender/motifect_ai_motion.zip` (run `python scripts/sync_all.py`)

**Checklist:**
- [x] GPL-3.0 LICENSE in add-on folder
- [x] SPDX headers on Python files
- [x] `blender_manifest.toml` (Blender 4.2+ extensions layout)
- [x] Respects `Allow Online Access`
- [ ] Creator application approved
- [ ] 5+ screenshots (2:1, 1200px+ wide)
- [ ] On-site documentation (install, API key, credits, usage)
- [ ] Publication date ≥ 2 weeks out

**Install steps for listing:**
1. Download zip → Blender Preferences → Add-ons → Install
2. Enable Motifect Motion → paste API key
3. N panel → Motifect → Generate Motion
